## ðŸ“Œ Usage

### Pack
``` shell 
squish pack ./my-folder -o archive.squish
```

### List
``` shell
squish list archive.squish
```
optional simpified format:
``` shell 
squish list archive.squish --simple
```

### Unpack
``` shell
squish unpack archive.squish -o ./output-dir
```
